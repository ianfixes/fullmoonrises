# Full Moonrises

This is a simple program to find the dates, times, location, and direction of full-moon moonrises.

You will need `node.js`.

Installation:
```console
$ npm install
```

Operation (year, latitude, longitude):
```console
$ node index.js 2020 41.8240 -71.4128
```
